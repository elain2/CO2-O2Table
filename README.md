# 🌊 CO₂ / O₂ Table — Freediving Dry Training

프리다이빙 드라이 트레이닝을 위한 **CO₂ 테이블 / O₂ 테이블 생성기 + 타이머** 웹앱입니다.

🔗 **[Live Demo → elain2.github.io/CO2-O2Table](https://elain2.github.io/CO2-O2Table)**

## 📋 주요 기능

- **CO₂ 테이블** — 숨 참기 시간 고정, 휴식 시간을 점차 줄여 이산화탄소 내성 향상
- **O₂ 테이블** — 휴식 시간 고정, 숨 참기 시간을 점차 늘려 최대 정지 시간 향상
- **PB 기반 자동 계산** — 본인의 스태틱 PB 입력 시 세트 구성 자동 생성
- **인터랙티브 타이머** — 호흡(💨) ↔ 참기(🫁) 페이즈 자동 전환
- **알림음** — 페이즈 전환 및 훈련 완료 시 Web Audio API 기반 사운드
- **시각화** — 세트별 참기/휴식 비율 바 차트

---

## 🚀 사용 방법

1. 본인의 **스태틱 PB** (분/초) 입력
2. **CO₂ 테이블** 또는 **O₂ 테이블** 선택
3. **GENERATE** 버튼으로 테이블 생성
4. **▶ 시작** 버튼으로 타이머 실행

별도 설치 없이 브라우저에서 바로 사용 가능합니다.

---

## 🧠 CO₂ vs O₂ 테이블

| | CO₂ 테이블 | O₂ 테이블 |
|---|---|---|
| **목적** | CO₂ 내성 향상 | 최대 정지 시간 향상 |
| **참기 시간** | 고정 (PB의 50%) | 점차 증가 |
| **휴식 시간** | 점차 감소 | 고정 (2분) |
| **체감 난이도** | 후반으로 갈수록 힘듦 | 후반으로 갈수록 힘듦 |
| **권장 시기** | 훈련 초반 / 주 2~3회 | CO₂ 테이블 적응 후 병행 |
